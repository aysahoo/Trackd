# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Trackd, please report it responsibly.

**Do not open a public issue for security vulnerabilities.**

Instead, please DM [@ayushfyi](https://x.com/ayushfyi) or [@adarshanatia](https://x.com/adarshanatia) on X.

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

### What to expect

- We'll try to acknowledge your report within 48 hours
- We'll try to investigate and keep you updated on our progress
- Once fixed, we'll try to credit you (unless you prefer to remain anonymous)

Thank you for helping keep Trackd secure!
